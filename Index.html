Semantic Elements

Semantic elements help the html page to be formatted properly.

<section>

Divides a web page into standalone sections of related content

<article>

Self contained composition intended to be independently distributable or reusable

<nav>

Section of a page that links to other parts of the same page

<div>

Defines a division / section within an HTML document that can be assigned a tag and styled with CSS

<aside>

Puts a section of HTML code aside on a page when it is of lesser importance

<span>

Groups elements together for styling purposes
